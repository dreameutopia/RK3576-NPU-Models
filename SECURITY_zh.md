## 报告漏洞

如果您在本项目中发现安全漏洞，请负责任地报告。

### 如何报告

1. **不要**为此类漏洞公开创建 GitHub Issue
2. 直接联系维护者或使用 [GitHub 私密漏洞报告](https://github.com/dreameutopia/RK3576-NPU-Models/security/advisories/new)
3. 请包含以下信息：
   - 漏洞描述
   - 复现步骤
   - 潜在影响
   - 修复建议（如有）

### 预期响应

- **48 小时内**确认收到
- **7 天内**更新处理状态
- **修复时间**取决于严重程度：
  - 严重（Critical）：7 天内
  - 高危（High）：14 天内
  - 中/低危（Medium/Low）：下个发布周期

## 安全使用建议

使用本项目时请注意：

- **模型文件**：仅从可信来源下载模型
- **Shell 脚本**：以提升权限运行前请先审查脚本内容
- **网络**：模型在本地运行，不会向外部服务器发送数据
- **依赖项**：保持 rknn-toolkit2 和系统软件包为最新版本
